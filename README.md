# GifOS

Proyecto front-end de GifOS para curso DWFS de ACAMICA que consitía en consumir API de giphy utilizando:

- HTML 
- CSS
- SASS
- JS

## Características 

- Se puede cambiar el tema de light a dark  
- Ver gif sugeridos y las tendencias
- Buscar y crear gifs
